tilehive - tiles gallery website

 purpose
tilehive is a simple website to showcase tiles for home decoration and construction. users can browse tiles, search them, check details of a tile, and manage their own profile after logging in.

 live url
https://ph-13-a08.onrender.com
key features
- home page with banner, marquee text and featured tiles
- all tiles page with search by title
- single tile details page (only visible after login)
- register and login with email and password
- login with google
- toast error message on wrong login/register
- my profile page, shows name, email and photo
- update name and photo from profile
- protected routes for tile details and my profile page
- loading spinner while data is fetching
- custom 404 not found page
- fully responsive on mobile, tablet and desktop

npm packages used
- next js
- react / react-dom
- tailwindcss
- daisyui
- swiper
- better-auth
- mongodb
- json-server
